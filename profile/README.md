## 리딩로그 (Reading Log)

리딩로그는 바쁜 현대인을 위한 스마트 독서 기록 및 관리 서비스입니다.
20~30대를 주요 타겟으로 하며, 독서 습관을 자연스럽게 형성할 수 있도록 돕습니다.

사용자는 독서 타임워치를 활용하여 실시간으로 읽은 시간을 기록하고, 목표에 맞춰 독서를 계획할 수 있습니다.
또한 책 목록을 관리하고, 월별·연별 독서 통계를 시각적으로 확인하여 성취감을 느낄 수 있습니다.
